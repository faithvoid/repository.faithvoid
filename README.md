# repository.faithvoid
faithvoid's XBMC4Xbox 4.0 add-on repository.

when opening issues, please specifically describe which add-on is causing issues in the title, ie; "nekoScraper - Doesn't grab synopsis data for XYZ region"

## nekoScraper
Scraper for XBMC4Gamers artwork and metadata, powered by IGDB + MobCat's Xbox Title ID Database.

## nekoFeatures
Feature list browser for XBMC4Xbox, meant to be used in combination with nekoScraper metadata.
